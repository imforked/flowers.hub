# Voice models

- **flowers.ppn** — Picovoice Porcupine wake word (“flowers”). Required. Create at [Picovoice Console](https://console.picovoice.ai/).

Command phrase recognition (“play messages”) uses **Picovoice Leopard** (speech-to-text). Same `PICOVOICE_KEY` as Porcupine; no separate model file. Install with `pip install pvleopard` (works on Raspberry Pi).
